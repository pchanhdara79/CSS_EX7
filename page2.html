display: flex method to create equal width buttons and tabs just for fun.

This menu is using display: flex to create horizontal display.

With flexbox, the unordered list is the flex container and the list items are the children.

Using equal flex values makes each tab the same width.

The current tab matches the page background color to create a “tab” effect.
